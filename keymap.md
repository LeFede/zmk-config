# Configuración ZMK - Corne 36 teclas

## **Capa 0: Base**
```
Q  W  E  R  T        Y  U  I  O  P
A  S  D  F  G        H  J  K  L  ;
Z  X  C  V  B        N  M  ,  .  /
      _  1  SPC      ENT  2  _
```
**Home row mods:**
- A=GUI, S=ALT, D=SHIFT, F=CTRL (izq)
- J=CTRL, K=SHIFT, L=ALT, ;=GUI (der)

**Pulgares:**
- 1 = Capa Numpad (hold)
- 2 = Capa Edit (hold)
- ENT = Enter (tap) / Capa Nav (hold)

---

## Notas
- `_` = transparente/vacío
- BT1-5 = perfiles Bluetooth (cambiar dispositivo)
- BTCLR = borrar todos los perfiles BT
- Home row mods: tap=letra, hold=modificador
